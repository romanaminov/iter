Подробное описание

В данном файле реализована модель динамики полоидальных токов в токамаке ITER. Меодель представляет собой систему линейных дифф. уравнений 1 порядка составленых по правилу Кирхгофа. Результатом являются два контейнера. В одном хранятся значения полоидальных токов в контурах в каждый момент времени, в другом - производные этих токов.
